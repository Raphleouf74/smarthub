Veuillez entrer le code à 6 chiffres de votre stylo connecté ou cherchez le via Bluetooth.

Qu'est-ce que Stylus Hub ?

Stylus Hub est un outil qui vous permet de voir les différentes informations de votre stylo connécté*, simplement, rapidement et gratuitement**. Vous pourrez voir le pourcentage d'encre restant dans le stylo, le nombre de pages qu'il peut écrire***, son niveau de batterie et bien plus. Qu'attendez-vous ? Allez vite en magasin ou sur notre boutique en ligne pour vous procurer ce stylo****.




*: Uniquement disponible pour les stylos connéctés avec le label "Compatible StylusHub".
**: Le site et l'application mobile sont gratuit mais le stylo doit être acheté en magasin ou sur notre boutique en ligne.
*** : Le nombre est approximatif, il peut varier dans des condition réelles. Complètement rechargé, le stylo peut écrire environ 120 pages.
Les conditions s'appliquent. Sous réserve de stocks disponibles.
